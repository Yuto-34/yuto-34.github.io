課題

前述のGauss-Jordan 法を用いて N 元一次連立方程式を解くプログラムを対象に，以下を実施せよ．

  1. 計算結果，途中結果(verbose=1のとき)を表示する場合は，有効数字を 5 桁の指数表記とする．
  2. ピボットの入れ替えが必要な行列や， 特異行列もしくはたちの悪い行列が与えられた場合は， その旨を検出して表示するようにせよ．
    「特異行列」：ピボットが0となる．(行列式が0となる場合，ピボットが0となる．運が悪いと，行列式が0でなくてもピボットは0となり得る．)
    「たちの悪い行列」:ここでは，各ピボットに対して同一行の他の成分が1000倍よりも大きくなる場合がどこかで生じる場合とする．
    これらへの対応：「各ピボットが0になる，或いは，各ピボットに対して同一行の他の成分が1000倍よりも大きくなる」ことが検出された場合，メッセージを出力してストップする．
    検出することさえできれば，それ以上の処理は特に行なわなくとも良い． テストデータ Simal04.dat，Simal05.dat，Simal10.dat などを確実に検出できること．
    (どのように修正したか考察にて明記すること)
  3. Simal08.datとSimal08m.dat に対して，停止することなく解が得られることを示す．
  4. Simal04.dat に対して，対角成分が0となることを検出して，停止することを示す．
  5. Simal05.datとSimal05m.dat に対して，ピボットに比べて他成分(係数)にが相対的に非常に大きくなることを検出して，停止することを示す(コードの修正が必要)．
  6. Simal10.dat ：現状では対角成分が0となり停止するはず．入力データを工夫することによって解が得られることを示す(コードの修正は要さない)．
  7. 各自が用意した連立多元一次方程式に対して実行して結果を示す．

    余裕があれば
  8. 逆行列の求め方について説明せよ．
